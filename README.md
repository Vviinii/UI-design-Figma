Smart Waste Collection App – Stadt Aschaffenburg
This repository contains a Figma-based UI concept for a smart waste collection app designed for the city of Aschaffenburg. The goal is to support more efficient, data-driven waste management and improve the experience for both citizens and collection teams.

Project Overview
The app focuses on making municipal waste collection more transparent and convenient by combining clear navigation with actionable information for residents and operations staff.

Key ideas:

Route and schedule overview for collection teams to reduce empty runs and optimize capacity.

Citizen view with pick-up dates, bin status, and notifications for upcoming collections.

Map-based interface for locating nearby containers and special collection points.

Design language tailored to Aschaffenburg’s visual identity and existing digital services.
